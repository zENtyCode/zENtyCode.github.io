【经验分享】从零搭建hexo博客

WIN10环境下从零搭建hexo博客

本教程基本全程参考B站UP主codesheep执行
B站av号：av44544186

hexo框架介绍

hexo是一个轻量高效的静态博客框架，自身比较成熟用的人比较多，遇到相关问题在网上寻找解决方法也比较容易，具体的hexo框架介绍请自行百度

相关软件安装

git安装

https://git-scm.com/

安装结束后进入cmd输入以下命令并回车

git --version

打印正确的版本号即安装成功

https://www.runoob.com/git/git-install-setup.html

node.js安装

https://nodejs.org/en/

【图片】点击这里下载LTS长期支持版本就可以，安装过程请自行百度
安装结束后进入cmd输入以下命令并回车

node -v

打印正确版本号即安装成功

这里其实是安装了两个东西，一个是node.js，另外一个是npm工具，等下我们会用到它

正式安装hexo博客框架

安装之前

换源，国内镜像源安装各种插件非常慢，cmd中键入以下命令并回车，等待进度条充满即安装完毕，如无反应请检查命令输入是否正确

npm install -g cnpm --registry=https://registry.npm.taobao.org

命令行输入

cnpm -v回车

打印版本号未报错即安装成功

安装hexo框架

命令行键入

cnpm install -g hexo-cli 回车

安装结束后用

hexo -v

进行验证

新建空文件夹安装hexo

在你认为合适的地方新建一个空文件夹，文件夹名无所谓，例如：blog

【图片】blog文件夹

打开文件夹，右键空白处，在右键菜单中点击Git Bash Here，弹出一个新的命令行窗口，之后的所有命令均在此窗口输入

【图片】git窗口，用户名打码

在空文件夹下执行

hexo init 回车

耐心等待

【图片】安装成功

此时hexo框架已经安装成功，命令行中键入

hexo s 回车

此命令常用于修改博客后进行本地的预览，并不会将你的改动更新到远端
打开浏览器，输入地址

localhost:4000 回车

浏览器中便可看到你的博客的样子，里面有一篇自动生成的helloworld文章

【图片】4000

博客网页的默认主题为landscape，可以自己更换其他的主题，预览状态下在git命令窗口按下ctrl+C退出预览

hexo基础命令及工作流

这里只介绍使用比较多的hexo指令，完整指令请参考hexo官方文档

https://hexo.io/zh-cn/docs/commands.html

hexo n "文章标题"

此命令新建一篇文章，格式为markdown，关于markdown是什么请自行百度，程序员的技术文章多为markdown格式，做笔记也非常适用，使用记事本、VSCode、markdownPad均可编辑md格式文件，笔者用的编辑器是Yu Writer
文章源文件保存在blog/source/_posts文件夹中，建议新建文章时使用此命令，不要直接进入文件夹新建

hexo clean

清理缓存文件，对博客进行更改或添加新的文章时进行清理是一个好的习惯

hexo g

g是generate的缩写，生成静态文件，对之前的所有改动在本地进行部署，此时可以使用hexo s命令在本地进行预览，查看改动的效果
下图为使用hexo n命令添加了两篇文章的效果图，可以发现刚才新建的博客文章已经出现在网页上了

【图片】

hexo远端部署

介绍一个通过github免费进行远端部署的方法，以后就通过github的地址来访问你的博客，当然自己花钱买一个域名也完全没有问题

接下来就来演示如何将博客部署到github上公开使用

登录github，在右上角的加号菜单下选择new repository，新建一个仓库

【图片】

给仓库命名一定是你的github昵称.github.io，必须是这个名字，下面描述我就写我的hexo博客，点击绿色的Create，这样就创建好了，不要着急关闭这个页面，记下浏览器地址栏中的地址

安装git部署插件

cnpm install --save hexo-deployer-git

【图片】安装成功

打开blog目录下的config.yml文件，建议用VS Code打开，文件结构简洁明了
到文件底部，找到# deployment，修改为以下样式

【图片】

repo里面是你的github仓库地址，也就是新建仓库后打开页面的地址栏中的地址，网页中也可以看到这个地址

初次运行git时的配置（重要）

需要对git进行配置，才能将博客部署到远端，详细说明见

https://git-scm.com/book/zh/v2/%E8%B5%B7%E6%AD%A5-%E5%88%9D%E6%AC%A1%E8%BF%90%E8%A1%8C-Git-%E5%89%8D%E7%9A%84%E9%85%8D%E7%BD%AE

打开git命令窗口，键入

git config --global user.name "你的git用户名"
git config --global user.email 你的邮件地址
例子：
git config --global user.name "zENty"
git config --global user.email zenty_97@qq.com

远端部署！

接下来万事俱备，我们就可以通过hexo命令来将你的博客直接部署到远端

使用hexo常规工作流，今后进行任何改动后建议使用此工作流

hexo clean
hexo g
hexo s //本地预览确认无误
hexo d //部署到远端

部署成功会提示INFO Deploy done ：git，接下来就可以通过浏览器访问你的博客了，例如：zentycode.github.io

这样一来hexo博客从安装到搭建再到推向远端就全部完成

关于hexo博客如何更换主题等其他问题请看笔者的其他文章
